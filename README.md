# tampermonkey-script

本项目用于存储一些自制油猴脚本

## 安装

1. 安装油猴插件：https://www.tampermonkey.net/
2. 进入 [src](src) 文件夹
3. 进入脚本文件，点 raw 按钮，油猴插件会自动识别脚本并安装

    需要先安装 [tm 环境](src/tm.user.js)，才能使用其他脚本
